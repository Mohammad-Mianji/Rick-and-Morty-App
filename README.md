# 🟡 PP-001 | Rick & Morty Explorer

A React application for exploring characters from the Rick and Morty universe.

This project was built as part of my React learning journey and focuses on working with REST APIs, asynchronous data fetching, searching, pagination, and component-based architecture.

---

## ✨ Features

- Browse Rick & Morty characters
- Search characters
- Pagination
- Character Details
- Favorite Characters
- Responsive UI

---

## 🛠️ Tech Stack

- React
- JavaScript (ES6+)
- CSS
- REST API

# 🇮🇷 توضیحات فارسی

## معرفی پروژه

این پروژه با استفاده از React توسعه داده شده و از API رسمی Rick & Morty برای دریافت اطلاعات شخصیت‌ها استفاده می‌کند.

هدف اصلی این پروژه، تمرین کار با REST API، مدیریت State، جستجو، صفحه‌بندی (Pagination) و ساخت رابط کاربری مبتنی بر کامپوننت بوده است.
